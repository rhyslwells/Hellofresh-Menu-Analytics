HelloFresh Menu Analytics

HelloFresh Menu Analytics is a lightweight weekly data pipeline that ingests HelloFresh API data, applies dimensional modeling with SCD Type 2 tracking, and generates comprehensive analytics reports.

All data is processed through a medallion architecture—from raw bronze ingestion through silver normalization to gold-layer analytics. Reports are generated automatically every week with visualizations and insights stored in SQLite.

What We Track:

  • Menu Changes: Weekly additions, removals, and modifications to recipes
  • Historical Data: Complete SCD Type 2 change history for all menu items
  • Recipe Metrics: Ingredient composition, nutrition, and allergen information
  • Trend Analysis: Cuisine distribution, ingredient popularity, and seasonal patterns
  • Data Quality: Automated validation and quality checks across all layers
